# Governance Analysis Dashboard

An interactive web-based dashboard for visualizing and analyzing governance and state-level data.

## 📁 Project Structure

| File | Description |
|------|-------------|
| `index.html` | Main entry point of the application |
| `governance.html` | Governance data visualization page |
| `state.html` | State-level analysis page |
| `script.js` | Core JavaScript logic |
| `state.js` | State-specific data and interactions |
| `styles.css` | Stylesheet for the application |
| `Overall_analysis.png` | Summary analysis image |

## 🚀 Getting Started

1. Clone the repository:
```bash
   git clone https://github.com/anshul23102/<repo-name>.git
```
2. Open `index.html` in your browser to launch the dashboard.
